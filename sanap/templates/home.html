The Virtual Library for Ecosystem Assessments in Europe

This Virtual Library is intended for uploading specific information on relevant resources for ecosystem assessment in Europe (including grey literature, tools, events, websites and maps, in any language).

The library will provide a search tool for

  • Resources (document, websites, maps, events) relevant for ecosystem assessment in Europe
  • Experts that produced these resources
  • Organisations that produced these resources
